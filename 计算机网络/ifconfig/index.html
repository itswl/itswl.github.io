参数解释

比如 查看 网卡 eth0

  1. RUNNING : 表示网络是连通的
  2. MTU : 默认为 1500
  3. TX : 发送
  4. RX : 接受
  5. errors : 错误的数据包数，eg: 校验错误，帧同步错误
  6. dropped : 丢弃的数据包数，数据包已经到了 ring buffer, 但因为内存不足等原因丢包
  7. overruns : 超限数据包， 网络 IO 过快，导致 ring buffer 中的数据包来不及处理（队列满）而导致的丢包
  8. carrier : 一般是双工不匹配，物理电缆出现问题
  9. collisions : 碰撞数据包数
