数据库操作

数据库操作基本围绕着对键值和目录的 CRUD 操作及其对应的生命周期管理。 etcd 这些操作符合 REST 风格的一套 API 操作

键操作

键操作包括最常用的增删改查操作，包括 PUT、GET、DELETE 等命令。

PUT 设置或者更新某个键的值

GET 获取指定键的值

DELETE 删除一个键或者特定范围的键

watch 键值对的改动

watch 监测一个或多个键值的变化，一旦键值发生更新，就会输出最新的值并退出

新开窗口

查看 watch 窗口

lease（租约）

一旦租约的 TTL 到期，租约就会过期并且所有附带的键都将被删除
